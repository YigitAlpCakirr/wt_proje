# Web Teknolojileri Projesi - Kişisel Portfolyo

Bu proje, Sakarya Üniversitesi Web Teknolojileri dersi kapsamında hazırlanmış bir kişisel web sitesidir. Site; hakkımda bilgiler, şehrim, kültürel mirasımız ve dinamik bir iletişim formu gibi çeşitli bölümlerden oluşmaktadır.

## 🚀 Canlı Önizleme
Projeye şu adresten ulaşabilirsiniz: **[https://yigitalp.rf.gd](https://yigitalp.rf.gd)**

---

## 🛠️ Kullanılan Teknolojiler

*   **Frontend:** HTML5, CSS3, Bootstrap 5, JavaScript
*   **Dinamik Yapı:** Vue.js (İletişim formu denetimi için kullanıldı)
*   **Backend:** PHP (Navbar yönetimi, Login sistemi ve Form işleme)
*   **API:** Rick and Morty API (İlgi alanlarım sayfasında karakter çekmek için kullanıldı)
*   **Animasyonlar:** AOS (Animate On Scroll) kütüphanesi

---

## 📌 Sayfalar ve Özellikler

1.  **Anasayfa:** Karşılama ekranı ve genel tanıtım.
2.  **Özgeçmişim (Hakkımda):** Eğitim ve yeteneklerimin yer aldığı bölüm.
3.  **Şehrim:** Memleketim Samsun hakkında bilgiler ve görsel galeri.
4.  **Mirasımız:** Kültürel değerlerimizi tanıtan içerik.
5.  **İlgi Alanlarım:** Rick and Morty API'sini kullanarak rastgele karakterler getiren dinamik sayfa.
6.  **İletişim:** 
    *   İki farklı butonla denetim sağlanmaktadır.
    *   **Native JS** ve **Vue.js** ile ayrı ayrı form validasyonu yapılmaktadır.
    *   Gönderilen veriler PHP tarafında (`iletisim.php`) karşılanıp listelenmektedir.
7.  **Login:** Öğrenci numarası ve e-posta kontrolü ile çalışan giriş sistemi.

---

## ⚙️ Kurulum ve Çalıştırma

Projeyi yerel bilgisayarınızda çalıştırmak için:
1. Bir PHP sunucusuna (XAMPP, WampServer vb.) ihtiyacınız vardır.
2. Dosyaları `htdocs` klasörüne kopyalayın.
3. `.htaccess` dosyasının dizinde olduğundan emin olun (HTML dosyalarında PHP çalışması için gereklidir).
4. `index.html` üzerinden siteyi görüntüleyebilirsiniz.
